# Smartshelf

Projekt "SmartShelf" ma na celu stworzenie nowoczesnej aplikacji internetowej do zarządzania osobistą kolekcją książek. Aplikacja pozwoli użytkownikom na tworzenie kont, katalogowanie swoich książek na wirtualnych półkach (np. "Do przeczytania", "Przeczytane"), śledzenie postępów w czytaniu oraz dodawanie osobistych notatek. Kluczową cechą systemu będzie "inteligentne" dodawanie książek – po podaniu numeru ISBN, system automatycznie pobierze szczegółowe informacje o książce (tytuł, autor, okładka, opis) z zewnętrznego, publicznego API, wykorzystując do tego zadania asynchroniczne w tle.
Projekt ten służy jako kompleksowe zadanie szkoleniowe, obejmujące cały cykl życia oprogramowania, od projektowania architektury, przez development full-stack, aż po konteneryzację i przygotowanie do wdrożenia.

Cele projektu

1. Cel funkcjonalny: Stworzenie w pełni funkcjonalnej, bezpiecznej i intuicyjnej platformy dla miłośników książek do zarządzania ich prywatnymi biblioteczkami.
   
2. Cel techniczny: Zbudowanie aplikacji w oparciu o nowoczesny stos technologiczny (Python/FastAPI, React, PostgreSQL, Docker), demonstrując umiejętność integracji różnych komponentów w spójną architekturę opartą na kontenerach.
   
3. Cel edukacyjny: Przećwiczenie kluczowych koncepcji inżynierii oprogramowania, takich jak:
   
- Projektowanie i implementacja API REST.
- Zarządzanie stanem w aplikacji front-endowej.
- Modelowanie relacyjnej bazy danych i użycie ORM.
- Implementacja systemu uwierzytelniania i autoryzacji (JWT).
- Obsługa zadań asynchronicznych w tle.
- Konteneryzacja i orkiestracja wieloserwisowej aplikacji.
